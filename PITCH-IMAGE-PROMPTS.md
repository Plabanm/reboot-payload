# TrustPass — Pitch Image Generation Prompts (Before / After Style)

> Ready-to-paste prompts for an image model (DALL·E 3, Midjourney, Adobe Firefly, Ideogram, etc.) to create **“TODAY vs FUTURE” before/after pitch graphics** in the style of the reference “One-Click Verified Mortgage Vault” infographic.
>
> **How to use:** paste a full prompt into your image model. For **Midjourney**, append the parameters noted at the bottom. For **DALL·E / Firefly / Ideogram**, paste as-is. Generate at **landscape 3:2** (≈1536×1024). Note: image models often misspell text — keep on-image words short, and regenerate or fix labels in Canva/Figma if needed.

---

## SHARED STYLE GUIDE (applies to every prompt)

Reference this look in all images:
- **Format:** wide landscape infographic, split into a **LEFT “TODAY” panel** and a **RIGHT “FUTURE” panel**, divided by a central arrow (→). A title bar across the top and a dark-green trust banner across the bottom.
- **Illustration style:** clean, modern **flat vector / corporate infographic** illustration. Soft rounded cards, subtle drop shadows, generous white space, friendly human characters in a simple flat style. NOT photorealistic, NOT 3D render — flat editorial vector, like a premium fintech explainer.
- **Colour system:**
  - LEFT / “before” panel = **soft red / blush pink** background (#FBEAE3 / #FCE9E2), red accents (#C0392B), red ✕ icons, a stressed character.
  - RIGHT / “after” panel = **mint / pale green** background (#E7F3EC), **deep Lloyds green** accents (#013528 / #006A4D), green ✓ check icons, a happy character giving a thumbs-up.
  - Tokens / digital assets = **gold/orange** (#D9842A) accents.
  - Headings deep green; body text dark grey.
- **Branding:** top-right corner shows a simple **black prancing-horse logo with “BANK”** (Lloyds-style; generic is fine). Bottom banner (deep green) reads **“Built on Trust. Powered by Digital Identity”** with three small icons + labels: *You own your data · You grant access · You stay in control*.
- **Typography:** bold sans-serif headings, clean readable labels. Numbered steps **1–4** down each panel inside small circular badges (red circles on the left, green circles on the right).
- **Tone:** trustworthy, premium, optimistic, enterprise-grade.

## TIPS FOR BEST RESULTS

- **Text accuracy:** image models frequently garble long on-image text. Keep labels short; expect to regenerate a few times, or generate the panels/icons and add final text in **Canva, Figma or PowerPoint** for a crisp pitch deck.
- **Consistency across the set:** reuse the **same style sentence** (“clean flat vector corporate infographic, deep-green / blush-red / gold palette, soft shadows, lots of white space”) in every prompt so all slides look like a family.
- **Aspect ratio:** 3:2 landscape for slides (1536×1024). For a tall poster, change to 4:5 or 2:3 and say “vertical infographic, stacked TODAY above FUTURE.”
- **Characters:** add “diverse, friendly, simple flat-illustrated characters” if you want inclusive personas.

### Midjourney parameter suffix (append to any prompt above)
`--ar 3:2 --style raw --v 6.1`
(For a flatter, more graphic look add `--no photo, 3d, realistic`.)

### DALL·E 3 / Firefly / Ideogram
Paste the prompt as-is. If text is messy, add: *“render all labels in clean bold sans-serif, spelled exactly as written.”*
